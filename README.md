## ðŸ˜Š How to run

Before starting, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed on your machine.

```sh
- Clone this repo:
$ git clone https://github.com/marcelopajr/moveit-next.git

- Enter directory:
$ cd moveit-next

- Install dependencies:
$ npm install

- Launch the app:
$ npm start
```
